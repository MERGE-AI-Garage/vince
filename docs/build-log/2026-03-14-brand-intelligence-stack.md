# Brand Intelligence Stack
## How Brand DNA Travels Through Vince

---

## Overview

Vince is a brand-aware creative production system. Every image generated, every piece of copy produced, every conversation Vince has is shaped by a five-layer intelligence stack that you build once and maintain over time. The layers build on each other — the output of each layer becomes input to the next.

```
┌─────────────────────────────────────────────┐
│  Layer 1: Brand DNA                         │  Raw intelligence
│  (website, images, documents)               │
└────────────────────┬────────────────────────┘
                     ↓
┌─────────────────────────────────────────────┐
│  Layer 2: Agent Directives                  │  Governance rules
│  (guardrails, forbidden combos, tone rules) │
└────────────────────┬────────────────────────┘
                     ↓
┌─────────────────────────────────────────────┐
│  Layer 3: Generation Prompt                 │  Creative brief
│  (9 sections synthesized from DNA + rules)  │  for every image
└────────────────────┬────────────────────────┘
                     ↓
┌─────────────────────────────────────────────┐
│  Layer 4: Quick Starters                    │  Reusable recipes
│  (prompt templates + copy briefs)           │  layered on top
└────────────────────┬────────────────────────┘
                     ↓
┌─────────────────────────────────────────────┐
│  Layer 5: Generation                        │  Output
│  (image, video, copy)                       │
└─────────────────────────────────────────────┘
```

The Vince chat agent (Vince the creative director) has access to **all layers simultaneously** during conversation — Layers 1–4 are all injected into its system instruction.

---

## Layer 1: Brand DNA

**What it is:** The raw brand intelligence — everything Vince knows about the brand extracted from real sources.

**Sources:**
- **Website analysis** — visual identity, messaging, positioning, product catalog
- **Image analysis** — photography style, color profile, composition patterns, lighting preferences
- **Document analysis** — brand standards, style guides, tone of voice docs, marketing playbooks

**What gets stored (`creative_studio_brand_profiles`):**
| Field | Contains |
|---|---|
| `visual_dna` | Photography style, lighting patterns, color palette, mood |
| `photography_style` | Camera preferences, depth of field, film stock, shooting style |
| `color_profile` | Primary/secondary/accent colors with hex values |
| `composition_rules` | Layout conventions, framing, spatial rules |
| `product_catalog` | Product names, descriptions, visual characteristics |
| `brand_identity` | Core values, positioning, personality, tagline |
| `tone_of_voice` | Personality descriptors, energy, formality, dos/don'ts |
| `typography` | Font style, text overlay approach |
| `brand_standards` | Writer guidelines, glossary, social voice, positioning framework |
| `brand_story` | Heritage, sustainability, values, customer promise |

**Key rule:** Never edit Layer 1 manually. Re-run the analysis functions to update it. Downstream layers use this as source of truth.

**Confidence score:** Each profile has a `confidence_score` (0–1). Low confidence means insufficient source material — add more images or documents before building downstream layers.

---

## Layer 2: Agent Directives

**What it is:** Structured governance rules extracted from the brand DNA. The compliance layer.

**Generated by:** `generate-brand-guardrails` edge function
**Stored in:** `creative_studio_agent_directives`

**Structure of each directive:**
```
Name:                  "Visual Identity Guardrails for AI Creative"
Persona:               "Brand identity specialist enforcing visual standards"
Focus Area:            visual_identity | photography | copy | compliance | general
Rules:                 6-10 rules with severity (error / warning / info)
Forbidden Combos:      3-5 element pairs that must never appear together
Required Elements:     3-5 things that must be present in specific contexts
Tone Guidelines:       2-3 sentences on voice and messaging guardrails
```

**Multiple directives per brand** — typically one per focus area:
- Visual Identity
- Photography & Composition
- AI Product Imagery Standards
- Typographic & Text Overlay Rules
- Brand Compliance & Regulatory
- Brand Voice & Messaging

**Important:** Directives are **inactive by default** when generated. You must review and activate them. Only active directives flow into Layer 3 and into Vince's chat context.

**When to regenerate:** After significant DNA changes. After activating, always re-synthesize Layer 3.

---

## Layer 3: Generation Prompt

**What it is:** A synthesized 9-section creative brief that gets prepended to every image generation request. This is what makes every image on-brand without the user having to repeat brand rules each time.

**Generated by:** `synthesize-generation-prompt` edge function
**Stored in:** `brand_generation_prompts`
**Input:** Brand DNA (Layer 1) + all **active** directives (Layer 2)

**The 9 sections:**
| # | Section | What it directs |
|---|---|---|
| 1 | **Brand Identity** | Core values, aesthetic positioning, tagline |
| 2 | **Tone of Voice** | Personality, energy, formality, visual tone dos/don'ts |
| 3 | **Visual Style** | Signature aesthetics, visual differentiators |
| 4 | **Color Palette** | Mandatory colors (hex), forbidden colors, palette rules |
| 5 | **Typography** | Font style, text overlay approach, legibility rules |
| 6 | **Photography Direction** | Aperture, focal length, lighting, depth of field, film stock |
| 7 | **Composition Rules** | Layouts, framing, aspect ratios, spatial arrangement |
| 8 | **Brand Guardrails** | Hard rules, forbidden elements, required elements |
| 9 | **Brand Story** | Heritage, values, sustainability — as visual creative direction |

**Section toggles:** Each section can be turned on/off per generation (`section_toggles` JSONB). Useful when certain sections conflict with a specific job (e.g., disable Brand Story for a clean product shot).

**Critical dependency:** If you activate new directives **after** synthesizing the generation prompt, the prompt is stale. The Brand Guardrails section won't include the new directives until you re-synthesize.

**Correct order:**
1. Generate + activate directives (Layer 2)
2. **Then** synthesize the generation prompt (Layer 3)

## Layer 4: Quick Starters (Prompt Templates)

**What it is:** Pre-built creative recipes that layer on top of the generation prompt. Each starter is a specific, reusable brief for a common creative scenario.

**Generated by:** `generate-brand-starters` edge function (the "Generate with Gemini" button)
**Stored in:** `creative_studio_brand_prompts`

**Two fundamentally different types:**

### Visual Starters
Direction for AI image and video generation. Combine with the generation prompt at execution time.

| Category | Purpose |
|---|---|
| `product` | Product hero shots, detail shots, packshots |
| `hero` | Brand hero images, headshots in scene |
| `lifestyle` | In-context product use, aspirational scenes |
| `editorial` | Magazine-style, art-directed compositions |
| `campaign` | Campaign concepts, themed series |
| `social` | Platform-specific (Instagram, LinkedIn, TikTok) |
| `cinematography` | Video direction, motion, scene composition |

**Prompt template:** 2–4 sentences of specific creative direction
**Camera preset:** Aperture, focal length, lighting, film stock, depth of field
**Variable fields:** Dropdowns for product, campaign theme, etc.

### Copy Starters
Complete structured creative briefs for writing. Used by copywriters or pasted into any AI writing tool.

**`copy` category** — A full agency creative brief (150–300 words):
```
COPY BRIEF: {{content_type}} for {{platform}}
TARGET AUDIENCE:     Specific persona with mindset and tension
AUDIENCE MIND STATE: What they're feeling before seeing this
OBJECTIVE:           Specific shift to achieve, not just "awareness"
SINGLE-MINDED PROP:  One irreducible truth to land
PROOF POINTS:        2-3 RTBs from brand DNA
VOICE DIRECTION:     Tone spectrum, write-like/not-like, vocabulary
FORMAT:              Word count, structure blueprint, mandatory inclusions
EMOTIONAL ARC:       Enters feeling X → exits feeling Y → does Z
DO NOT:              Hard no-gos from brand governance
```
**Variable fields:** Platform (select), content type (select), audience segment (select), tone variation (select)

**`brand-voice` category** — A deployable AI system prompt (300–500 words). Paste into Gemini, Claude, or ChatGPT before any writing session to prime it with the brand's full voice identity:
```
You are a copywriter for [Brand]...

BRAND VOICE:         Core voice descriptor
VOICE PILLARS:       3 named pillars, each with ✓/✗ example sentences
VOCABULARY:          10-15 use / 10-15 avoid
GRAMMAR & STYLE:     5-7 specific rules
TONE BY CHANNEL:     Social / LinkedIn / Email / Long-form / OOH
YOU ARE WRITING:     {{content_type}} for {{audience}}
POSITIONING ANCHOR:  The strategic stake every sentence connects back to
```
**Variable fields:** Content type (select), audience (select), platform (select)

**Independence:** Quick Starters do not go stale when you regenerate Layers 1–3. They are frozen when created. Delete and regenerate to refresh.

## Layer 5: Generation

**What it is:** The actual creative output — images, video, or copy.

**How image generation works:**
```
Final prompt = Generation Prompt (Layer 3, filtered by section toggles)
             + Quick Starter template text (Layer 4, with variables resolved)
             + User's specific direction (from chat or Director Mode)
             + Camera preset parameters
             + Reference collections (if set)
```

**How copy generation works:**
```
Copy brief = Quick Starter copy brief (Layer 4, with variables resolved)
           → User pastes into Vince chat or external AI tool
           → AI writes against the brief
```

## The Vince Chat Agent

Vince the creative director has a different context than the generation pipeline. When a conversation starts, Vince loads:

- Full brand DNA (all profile fields from Layer 1)
- All **active** directives (Layer 2)
- Top 20 most-used prompt templates by usage count (Layer 4)
- Brand basics: name, voice, visual identity statement

Vince does **not** use the Generation Prompt (Layer 3) in conversation — that's a generation-time artifact. Vince builds its own creative direction from the raw DNA and directives, then presents prompts for user confirmation before generating.

## Setup Order for a New Brand

```
1. ANALYZE SOURCES
   ├── Analyze website (website analysis function)
   ├── Upload + analyze brand images (image analysis)
   └── Upload + analyze brand documents (doc analysis)

2. SYNTHESIZE PROFILE
   └── Merge all intelligence → brand profile (Layer 1 complete)

3. GENERATE DIRECTIVES
   ├── Run generate-brand-guardrails
   ├── Review each directive
   └── Activate the ones you want enforced (Layer 2 complete)

4. SYNTHESIZE GENERATION PROMPT
   └── Run synthesize-generation-prompt
       (reads active directives — do this AFTER step 3)
       (Layer 3 complete)

5. GENERATE QUICK STARTERS
   └── Click "Generate with Gemini"
       (reads DNA + directives — do this AFTER step 3)
       Expect: 14-20 starters including 2+ copy briefs + 2+ brand-voice prompts
       (Layer 4 complete)

6. GENERATE
   └── Use Director Mode or Vince chat
```

## Maintenance: When to Regenerate What

| Change | What to regenerate |
|---|---|
| New brand images analyzed | Re-synthesize profile → re-synthesize generation prompt → optionally refresh starters |
| New brand document uploaded | Re-synthesize profile → re-synthesize generation prompt |
| Added/changed a directive | Re-synthesize generation prompt only |
| Deactivated a directive | Re-synthesize generation prompt only |
| Want better copy starters | Delete auto-generated starters → re-run Generate with Gemini |
| Changed brand voice/tone | Re-synthesize profile → re-generate directives → re-synthesize generation prompt → refresh starters |

**Starters are independent** — they don't cascade. Changing Layer 1–3 does not automatically update your starters. You decide when to refresh them.
